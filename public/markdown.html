Code and Syntax Highlighting

Tables

Tables don’t look great right now, but you can use them. Colons can be used to align columns.

| Tables     |          Hold | Data |
| ---------- | ------------: | :--: |
| col 3 is   | right-aligned | 123  |
| col 2 is   |      centered | 456  |
| some other |          text | 789  |
